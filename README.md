# Thanh Hóa Land

Landing page bất động sản đơn giản, responsive, xây dựng bằng Next.js và TypeScript.

## Chạy local

```bash
npm install
npm run dev
```

Mở http://localhost:3000

## Build production

```bash
npm run build
npm start
```

## Cần chỉnh trước khi public

- Thay số điện thoại `0912345678` trong `src/app/page.tsx`
- Thay dữ liệu bất động sản mẫu
- Thay logo, ảnh và thông tin liên hệ
- Kết nối domain sau khi deploy Vercel
